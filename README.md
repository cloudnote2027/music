# 🎵 S 0 S MODS - Premium RGB Music Player

ဒီ Project ကတော့ **Appwrite Cloud Database** ကို အသုံးပြုပြီး တည်ဆောက်ထားတဲ့ Modern RGB Music Player တစ်ခုဖြစ်ပါတယ်။ စာသား (Lyrics) များပါဝင်ပြီး ဖုန်းနဲ့ကြည့်ရင် ပိုမိုသပ်ရပ်အောင် **Mobile-First Design** နဲ့ ဖန်တီးထားပါတယ်။

---

## 🚀 Features (ထူးခြားချက်များ)

* **Full RGB Text Animation:** စာလုံးများအားလုံး အရောင်ပြေးနေတဲ့ Neon Style။
* **Cloud Database Sync:** Appwrite ကိုသုံးထားလို့ သီချင်းအသစ်တွေကို Database ကနေ တိုက်ရိုက် Update လုပ်နိုင်ခြင်း။
* **Lyrics System:** သီချင်းအချိန်အလိုက် စာသားများ အလိုအလျောက် ပေါ်လာခြင်း။
* **Scrollable Song List:** သီချင်းစာရင်းကို သီးသန့် Scroll ဆွဲပြီး ရွေးချယ်နိုင်ခြင်း။
* **Developer Identity:** S 0 S MODS ရဲ့ ကိုယ်ပိုင် Identity နှင့် Neon Glow Effects များ။

---

## ⚙️ Setup & Usage (အသုံးပြုပုံ)

၁။ ဒီ Repository ကို **Clone** လုပ်ပါ သို့မဟုတ် **Download** ဆွဲပါ။
၂။ `index.html` ကို Browser မှာ ဖွင့်ပါ။
၃။ Appwrite Project ID နှင့် Database IDs များ မှန်ကန်ရန် လိုအပ်ပါသည်။ (ကိုမျိုးရဲ့ ကိုယ်ပိုင် Project ဖြစ်လို့ Permissions တွေကို Dashboard မှာ Role: Any ပေးထားရန် လိုပါမယ်)
